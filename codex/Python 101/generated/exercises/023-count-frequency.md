# 023. Đếm tần suất xuất hiện

        ## Module

        List

        ## Difficulty

        Medium

        ## Exercise Type

        Thống kê bằng dictionary

        ## Prerequisites

        - List
- Dictionary
- Vòng lặp

        ## Python Topics

        - dict
- get()
- sorted()

        ## Math Or Reasoning Topics

        - Tần suất xuất hiện

        ## Problem Statement

        Nhập n và n số nguyên, đếm số lần xuất hiện của từng giá trị. In các giá trị theo thứ tự tăng dần, mỗi dòng gồm giá trị và số lần xuất hiện.

        ## Input

        Dòng thứ nhất là số nguyên n. Dòng thứ hai gồm n số nguyên.

        ## Output

        Nhiều dòng, mỗi dòng gồm value count theo thứ tự value tăng dần.

        ## Example

        ### Sample Input

        ```text
        7
2 3 2 1 3 2 1
        ```

        ### Sample Output

        ```text
        1 2
2 3
3 2
        ```

        ### Explanation

        Số 1 xuất hiện 2 lần, số 2 xuất hiện 3 lần, số 3 xuất hiện 2 lần.
