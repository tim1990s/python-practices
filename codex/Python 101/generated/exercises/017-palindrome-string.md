# 017. Kiểm tra chuỗi palindrome

        ## Module

        String

        ## Difficulty

        Easy-Medium

        ## Exercise Type

        So sánh chuỗi

        ## Prerequisites

        - Chuỗi
- Cắt chuỗi
- So sánh

        ## Python Topics

        - lower()
- isalnum()
- join()
- slicing

        ## Math Or Reasoning Topics

        - Tính đối xứng

        ## Problem Statement

        Nhập một chuỗi, bỏ qua khoảng trắng, dấu câu và hoa thường, kiểm tra chuỗi có phải palindrome hay không.

        ## Input

        Một dòng văn bản.

        ## Output

        In YES nếu là palindrome, ngược lại in NO.

        ## Example

        ### Sample Input

        ```text
        A man, a plan, a canal: Panama
        ```

        ### Sample Output

        ```text
        YES
        ```

        ### Explanation

        Sau khi chuẩn hóa còn amanaplanacanalpanama, chuỗi này đọc xuôi và ngược giống nhau.
