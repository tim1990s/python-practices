# 001. Tổng hai số

        ## Module

        Basic Math

        ## Difficulty

        Easy

        ## Exercise Type

        Tính toán cơ bản

        ## Prerequisites

        - Biến
- Số nguyên
- input()
- print()

        ## Python Topics

        - int()
- split()
- toán tử +

        ## Math Or Reasoning Topics

        - Phép cộng số nguyên

        ## Problem Statement

        Nhập hai số nguyên a và b, sau đó in ra tổng của chúng.

        ## Input

        Một dòng gồm hai số nguyên a và b, cách nhau bởi dấu cách.

        ## Output

        Một số nguyên duy nhất là a + b.

        ## Example

        ### Sample Input

        ```text
        3 5
        ```

        ### Sample Output

        ```text
        8
        ```

        ### Explanation

        Tổng của 3 và 5 là 8.
