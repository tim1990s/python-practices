# 009. Phân loại điểm số

        ## Module

        Condition If Else

        ## Difficulty

        Easy-Medium

        ## Exercise Type

        Phân loại theo khoảng

        ## Prerequisites

        - Số thực
- So sánh
- elif

        ## Python Topics

        - float()
- if
- elif
- else

        ## Math Or Reasoning Topics

        - Khoảng giá trị

        ## Problem Statement

        Nhập điểm số từ 0 đến 10 và phân loại kết quả học tập.

        ## Input

        Một số thực score.

        ## Output

        In INVALID nếu điểm ngoài khoảng 0 đến 10; EXCELLENT nếu điểm từ 8.5; GOOD nếu từ 7.0; AVERAGE nếu từ 5.0; ngược lại BELOW AVERAGE.

        ## Example

        ### Sample Input

        ```text
        8.7
        ```

        ### Sample Output

        ```text
        EXCELLENT
        ```

        ### Explanation

        8.7 nằm trong khoảng từ 8.5 đến 10 nên được phân loại EXCELLENT.
