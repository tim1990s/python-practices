# 029. Hàm tìm kiếm tuyến tính

        ## Module

        Function

        ## Difficulty

        Easy-Medium

        ## Exercise Type

        Hàm xử lý list

        ## Prerequisites

        - Hàm
- List
- Vòng lặp

        ## Python Topics

        - def
- list[int]
- enumerate()
- return

        ## Math Or Reasoning Topics

        - Tìm kiếm trong dãy hữu hạn

        ## Problem Statement

        Nhập n, list gồm n số nguyên và giá trị target, viết hàm tìm vị trí xuất hiện đầu tiên của target. Nếu không có thì trả về -1.

        ## Input

        Dòng thứ nhất là n. Dòng thứ hai gồm n số nguyên. Dòng thứ ba là target.

        ## Output

        Một số nguyên là chỉ số đầu tiên theo 0-based index, hoặc -1 nếu không tìm thấy.

        ## Example

        ### Sample Input

        ```text
        5
4 8 15 16 23
15
        ```

        ### Sample Output

        ```text
        2
        ```

        ### Explanation

        Giá trị 15 xuất hiện đầu tiên tại chỉ số 2.
