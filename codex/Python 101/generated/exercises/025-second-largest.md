# 025. Tìm số lớn thứ hai

        ## Module

        List

        ## Difficulty

        Medium

        ## Exercise Type

        Tìm cực trị có điều kiện

        ## Prerequisites

        - List
- Set
- Sắp xếp
- Điều kiện

        ## Python Topics

        - set()
- sorted()
- reverse=True

        ## Math Or Reasoning Topics

        - Thứ hạng trong dãy số

        ## Problem Statement

        Nhập n và n số nguyên, tìm giá trị lớn thứ hai khác với giá trị lớn nhất.

        ## Input

        Dòng thứ nhất là số nguyên n. Dòng thứ hai gồm n số nguyên.

        ## Output

        In số lớn thứ hai nếu tồn tại, ngược lại in NO SECOND LARGEST.

        ## Example

        ### Sample Input

        ```text
        6
10 5 10 8 7 8
        ```

        ### Sample Output

        ```text
        8
        ```

        ### Explanation

        Các giá trị khác nhau là 10, 8, 7, 5; số lớn thứ hai là 8.
