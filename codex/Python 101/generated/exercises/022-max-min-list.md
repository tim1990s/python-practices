# 022. Tìm giá trị nhỏ nhất và lớn nhất trong list

        ## Module

        List

        ## Difficulty

        Easy

        ## Exercise Type

        Duyệt list và so sánh

        ## Prerequisites

        - List
- So sánh
- for

        ## Python Topics

        - list
- min()
- max()

        ## Math Or Reasoning Topics

        - Giá trị cực trị trong dãy

        ## Problem Statement

        Nhập n và n số nguyên, in ra giá trị nhỏ nhất và lớn nhất.

        ## Input

        Dòng thứ nhất là số nguyên dương n. Dòng thứ hai gồm n số nguyên.

        ## Output

        Hai số nguyên: nhỏ nhất và lớn nhất.

        ## Example

        ### Sample Input

        ```text
        6
4 -1 9 9 2 0
        ```

        ### Sample Output

        ```text
        -1 9
        ```

        ### Explanation

        Số nhỏ nhất là -1 và số lớn nhất là 9.
