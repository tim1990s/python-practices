# 007. Tìm số nhỏ nhất và lớn nhất trong ba số

        ## Module

        Condition If Else

        ## Difficulty

        Easy

        ## Exercise Type

        So sánh nhiều giá trị

        ## Prerequisites

        - Số nguyên
- So sánh
- if elif else

        ## Python Topics

        - min()
- max()
- split()

        ## Math Or Reasoning Topics

        - Thứ tự trên tập số nguyên

        ## Problem Statement

        Nhập ba số nguyên a, b, c, sau đó in ra số nhỏ nhất và số lớn nhất.

        ## Input

        Một dòng gồm ba số nguyên a, b và c.

        ## Output

        Hai số nguyên: giá trị nhỏ nhất và giá trị lớn nhất.

        ## Example

        ### Sample Input

        ```text
        4 -2 9
        ```

        ### Sample Output

        ```text
        -2 9
        ```

        ### Explanation

        Trong ba số 4, -2 và 9, nhỏ nhất là -2 và lớn nhất là 9.
