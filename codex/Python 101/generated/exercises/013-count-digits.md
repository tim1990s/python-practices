# 013. Đếm số chữ số

        ## Module

        Loop

        ## Difficulty

        Easy-Medium

        ## Exercise Type

        Vòng lặp while

        ## Prerequisites

        - Số nguyên
- Phép chia nguyên
- while

        ## Python Topics

        - while
- abs()
- toán tử //

        ## Math Or Reasoning Topics

        - Biểu diễn thập phân của số nguyên

        ## Problem Statement

        Nhập một số nguyên n, đếm số chữ số trong biểu diễn thập phân của n.

        ## Input

        Một số nguyên n.

        ## Output

        Một số nguyên là số chữ số của n.

        ## Example

        ### Sample Input

        ```text
        -2030
        ```

        ### Sample Output

        ```text
        4
        ```

        ### Explanation

        Bỏ dấu âm, số 2030 có 4 chữ số.
