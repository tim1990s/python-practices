# 005. Kiểm tra số chính phương

        ## Module

        Basic Math

        ## Difficulty

        Easy-Medium

        ## Exercise Type

        Kiểm tra tính chất số học

        ## Prerequisites

        - Số nguyên
- Căn bậc hai
- So sánh

        ## Python Topics

        - int()
- math.isqrt()
- if đơn giản

        ## Math Or Reasoning Topics

        - Số chính phương
- Căn bậc hai nguyên

        ## Problem Statement

        Nhập một số nguyên không âm n, kiểm tra n có phải là số chính phương hay không.

        ## Input

        Một số nguyên không âm n.

        ## Output

        In YES nếu n là số chính phương, ngược lại in NO.

        ## Example

        ### Sample Input

        ```text
        49
        ```

        ### Sample Output

        ```text
        YES
        ```

        ### Explanation

        49 = 7 * 7 nên 49 là số chính phương.
