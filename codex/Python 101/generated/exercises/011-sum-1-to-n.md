# 011. Tính tổng từ 1 đến n

        ## Module

        Loop

        ## Difficulty

        Easy

        ## Exercise Type

        Tích lũy tổng

        ## Prerequisites

        - Số nguyên dương
- Vòng lặp for
- Biến tổng

        ## Python Topics

        - for
- range()
- biến tích lũy

        ## Math Or Reasoning Topics

        - Tổng cấp số cộng đơn giản

        ## Problem Statement

        Nhập số nguyên dương n, tính tổng các số từ 1 đến n.

        ## Input

        Một số nguyên dương n.

        ## Output

        Một số nguyên là tổng 1 + 2 + ... + n.

        ## Example

        ### Sample Input

        ```text
        5
        ```

        ### Sample Output

        ```text
        15
        ```

        ### Explanation

        1 + 2 + 3 + 4 + 5 = 15.
