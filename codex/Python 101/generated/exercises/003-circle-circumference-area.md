# 003. Chu vi và diện tích hình tròn

        ## Module

        Basic Math

        ## Difficulty

        Easy-Medium

        ## Exercise Type

        Tính toán với số thực

        ## Prerequisites

        - Số thực
- Hằng số pi
- Định dạng số

        ## Python Topics

        - float()
- math.pi
- f-string
- định dạng .2f

        ## Math Or Reasoning Topics

        - Chu vi hình tròn
- Diện tích hình tròn

        ## Problem Statement

        Nhập bán kính r của hình tròn, in ra chu vi và diện tích, mỗi giá trị làm tròn 2 chữ số thập phân.

        ## Input

        Một số thực dương r.

        ## Output

        Hai số thực: chu vi và diện tích, cách nhau bởi dấu cách, làm tròn 2 chữ số thập phân.

        ## Example

        ### Sample Input

        ```text
        2
        ```

        ### Sample Output

        ```text
        12.57 12.57
        ```

        ### Explanation

        Với r = 2, chu vi = 2 * pi * r và diện tích = pi * r * r, cả hai xấp xỉ 12.57.
