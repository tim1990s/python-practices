# 014. Kiểm tra số nguyên tố

        ## Module

        Loop

        ## Difficulty

        Medium

        ## Exercise Type

        Kiểm tra ước số

        ## Prerequisites

        - Số nguyên
- Ước số
- Vòng lặp
- if

        ## Python Topics

        - for
- range()
- math.isqrt()
- break

        ## Math Or Reasoning Topics

        - Số nguyên tố
- Ước số không tầm thường

        ## Problem Statement

        Nhập số nguyên n, kiểm tra n có phải là số nguyên tố hay không.

        ## Input

        Một số nguyên n.

        ## Output

        In YES nếu n là số nguyên tố, ngược lại in NO.

        ## Example

        ### Sample Input

        ```text
        29
        ```

        ### Sample Output

        ```text
        YES
        ```

        ### Explanation

        29 chỉ có hai ước dương là 1 và 29.
