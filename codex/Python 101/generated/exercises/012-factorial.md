# 012. Tính giai thừa

        ## Module

        Loop

        ## Difficulty

        Easy-Medium

        ## Exercise Type

        Tích lũy tích

        ## Prerequisites

        - Số nguyên không âm
- Vòng lặp for
- Phép nhân

        ## Python Topics

        - for
- range()
- biến product

        ## Math Or Reasoning Topics

        - Giai thừa

        ## Problem Statement

        Nhập số nguyên không âm n, tính n!.

        ## Input

        Một số nguyên không âm n.

        ## Output

        Một số nguyên là n!.

        ## Example

        ### Sample Input

        ```text
        5
        ```

        ### Sample Output

        ```text
        120
        ```

        ### Explanation

        5! = 1 * 2 * 3 * 4 * 5 = 120.
