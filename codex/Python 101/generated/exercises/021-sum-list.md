# 021. Tính tổng các phần tử trong list

        ## Module

        List

        ## Difficulty

        Easy

        ## Exercise Type

        Duyệt list và tích lũy

        ## Prerequisites

        - List
- Vòng lặp for
- Biến tổng

        ## Python Topics

        - list
- map()
- sum()

        ## Math Or Reasoning Topics

        - Tổng của dãy số

        ## Problem Statement

        Nhập n và n số nguyên, tính tổng các phần tử trong list.

        ## Input

        Dòng thứ nhất là số nguyên n. Dòng thứ hai gồm n số nguyên.

        ## Output

        Một số nguyên là tổng các phần tử.

        ## Example

        ### Sample Input

        ```text
        5
1 2 3 4 5
        ```

        ### Sample Output

        ```text
        15
        ```

        ### Explanation

        Tổng của 1, 2, 3, 4, 5 là 15.
