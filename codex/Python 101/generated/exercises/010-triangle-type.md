# 010. Phân loại tam giác

        ## Module

        Condition If Else

        ## Difficulty

        Medium

        ## Exercise Type

        Kiểm tra điều kiện hình học

        ## Prerequisites

        - Bất đẳng thức tam giác
- if elif else
- and

        ## Python Topics

        - int()

- elif

        ## Math Or Reasoning Topics

        - Bất đẳng thức tam giác
- Tam giác đều

        ## Problem Statement

        Nhập ba cạnh của một tam giác, kiểm tra chúng có tạo thành tam giác không và phân loại tam giác.

        ## Input

        Một dòng gồm ba số nguyên dương a, b và c.

        ## Output

        In NOT TRIANGLE nếu không tạo thành tam giác; EQUILATERAL nếu tam giác đều; ISOSCELES nếu tam giác cân; SCALENE nếu tam giác thường.

        ## Example

        ### Sample Input

        ```text
        5 5 8
        ```

        ### Sample Output

        ```text
        ISOSCELES
        ```

        ### Explanation

        Ba cạnh 5, 5, 8 tạo thành tam giác và có hai cạnh bằng nhau.
