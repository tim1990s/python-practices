# 030. Hàm giải phương trình bậc hai

        ## Module

        Function

        ## Difficulty

        Medium

        ## Exercise Type

        Hàm giải toán theo trường hợp

- Số thực

        ## Python Topics

        - def
- tuple
- math.sqrt()
- if elif else

        ## Math Or Reasoning Topics

        - Phương trình bậc hai
- Biệt thức delta

        ## Problem Statement

        Nhập ba hệ số a, b, c của phương trình ax^2 + bx + c = 0, viết hàm phân loại và tính nghiệm thực nếu có.

        ## Input

        Một dòng gồm ba số thực a, b và c.

        ## Output

        Nếu vô số nghiệm in Infinite solutions. Nếu vô nghiệm in No solution hoặc No real roots tùy trường hợp. Nếu có một nghiệm in One root: x = value. Nếu có hai nghiệm in Two roots: x1 = value, x2 = value. Các nghiệm làm tròn 2 chữ số thập phân.

        ## Example

        ### Sample Input

        ```text
        1 -3 2
        ```

        ### Sample Output

        ```text
        Two roots: x1 = 1.00, x2 = 2.00
        ```

        ### Explanation

        Delta = 1, hai nghiệm thực là 1 và 2.
