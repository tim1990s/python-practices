# 002. Diện tích hình chữ nhật

        ## Module

        Basic Math

        ## Difficulty

        Easy

        ## Exercise Type

        Áp dụng công thức hình học

        ## Prerequisites

        - Biến
- Số nguyên
- Phép nhân

        ## Python Topics

        - int()
- split()
- toán tử *

        ## Math Or Reasoning Topics

        - Diện tích hình chữ nhật

        ## Problem Statement

        Nhập chiều dài và chiều rộng của một hình chữ nhật, sau đó in ra diện tích.

        ## Input

        Một dòng gồm hai số nguyên dương length và width.

        ## Output

        Một số nguyên là diện tích hình chữ nhật.

        ## Example

        ### Sample Input

        ```text
        7 4
        ```

        ### Sample Output

        ```text
        28
        ```

        ### Explanation

        Diện tích bằng chiều dài nhân chiều rộng: 7 * 4 = 28.
