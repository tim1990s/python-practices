# 003. Chu vi và diện tích hình tròn - Examples

## Sample Input

```text
2
```

## Sample Output

```text
12.57 12.57
```

## Explanation

Với r = 2, chu vi = 2 * pi * r và diện tích = pi * r * r, cả hai xấp xỉ 12.57.
