# OOP Python - Tài Liệu Và Bài Tập Thực Hành

Thư mục này hệ thống hóa kiến thức lập trình hướng đối tượng trong Python từ cơ bản đến nâng cao.

## Cấu Trúc

- `docs/oop_python_guide.md`: tài liệu lý thuyết đầy đủ về OOP Python.
- `examples/oop_examples.py`: các ví dụ Python có thể chạy trực tiếp.
- `exercises/README.md`: 10 bài tập OOP tăng dần độ khó, chỉ có yêu cầu và gợi ý.
- `exercises/*.py`: file starter để tự luyện, không có lời giải sẵn.

## Cách Học Đề Xuất

1. Đọc `docs/oop_python_guide.md` theo thứ tự.
2. Chạy các ví dụ:

```bash
python examples/oop_examples.py
```

3. Làm bài tập trong `exercises/` từ bài 01 đến bài 10.
4. Sau khi làm xong mỗi bài, tự kiểm tra:
   - Class này đại diện cho thực thể nào?
   - Thuộc tính nào thuộc về object?
   - Hành vi nào nên là method?
   - Có cần dùng kế thừa không, hay composition phù hợp hơn?

## Lộ Trình 7 Ngày

- Ngày 1: Class, object, `__init__`, `self`; làm bài `01_student.py`.
- Ngày 2: Instance attribute, class attribute, method; làm bài `03_shapes.py`.
- Ngày 3: Encapsulation, private/protected, `@property`; làm bài `02_bank_account.py`.
- Ngày 4: Inheritance, overriding, `super()`; làm bài `05_employee_management.py`.
- Ngày 5: Polymorphism, abstraction, ABC; nâng cấp bài shapes hoặc game.
- Ngày 6: Composition, aggregation, dataclass, magic methods; làm bài cart/order.
- Ngày 7: Làm mini project `10_school_management.py`.
